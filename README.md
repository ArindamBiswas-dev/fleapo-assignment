# Getting Started

Clone this repo to local
```sh
git clone https://github.com/ArindamBiswas-dev/fleapo-assignment.git
```

## Step 1: Start Metro

After cloning to you local machine, follow these setps - 

```sh
cd fleapo-assignment
npm i
npm start
```
## Run on Android
```sh
npm run android
```

## Run on IOS
```sh
npm run ios
```
